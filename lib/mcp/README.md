# MCP (Model Context Protocol)

Ce module expose des capacités métier sous forme de tools structurés et
interopérables. L’objectif est de rendre le backend pilotable par des agents IA
de manière sûre et auditée.

Endpoints :
- `GET /api/mcp` : liste des tools disponibles
- `POST /api/mcp` : exécution d’un tool
